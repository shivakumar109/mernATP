### Backend Development ###

1. Create git repo
     git init
2. Add .gitignore file
3. Create .env file environment Variables & Read data  from .env with "dotenv" module
     npm install dotenv
4. Generate pakage,json
     npm init -y
5. Create express app
6. Connect to database
7. Add middleware { body parser ,error handlling middleware}
8. Design Schema and create Models
9. Design REST APIs for All resources
